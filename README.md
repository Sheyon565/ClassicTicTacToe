# Tic Tac Toe Game

A classic Tic Tac Toe game built with React and Tailwind CSS.

## Features
- Two-player gameplay
- Winner detection with highlighted squares
- Draw detection
- Reset button
- Responsive design

## Technologies
- React
- Tailwind CSS
- Vite
